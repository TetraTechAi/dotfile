# WezTerm

GPU加速対応のモダンなターミナルエミュレータ。

## 基本設定

| 設定 | 値 |
|------|-----|
| カラースキーム | Dracula+ |
| フォント | JetBrains Mono (12pt) |
| 背景透明度 | 85% |
| スクロールバック | 100,000行 |
| GPU | WebGpu有効 |

## キーバインド

### タブ操作

| キー | 動作 |
|------|------|
| `Cmd+t` | 新しいタブを開く |
| `Cmd+w` | タブを閉じる |
| `Cmd+Shift+[` | 前のタブ |
| `Cmd+Shift+]` | 次のタブ |
| `Cmd+1-9` | タブ番号で移動 |

### ペイン操作

| キー | 動作 |
|------|------|
| `Cmd+d` | 水平分割 |
| `Cmd+Shift+d` | 垂直分割 |
| `Cmd+Shift+h/j/k/l` | ペイン間移動 |
| `Cmd+Shift+矢印` | ペインサイズ調整 |
| `Cmd+Shift+w` | ペインを閉じる |
| `Cmd+z` | ペインを最大化/復元 |
| `Cmd+=` | ペインを水平に均等化 |
| `Cmd+Shift+=` | ペインを垂直に均等化 |

### その他

| キー | 動作 |
|------|------|
| `Cmd+Enter` | フルスクリーン切替 |
| `Cmd+b` | 背景切替（透明 ↔ blur） |
| `Shift+Enter` | 改行送信（複数行入力用） |

## 特徴

- **リガチャ対応**: プログラミング用合字を表示
- **日本語入力**: IME有効
- **自動リロード**: 設定変更を自動反映
- **タブバー**: タブが1つの場合は非表示
