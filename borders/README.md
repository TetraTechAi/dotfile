# Borders

ウィンドウに色付きボーダーを表示するツール。アクティブウィンドウを視覚的に識別しやすくする。

## 設定

| 設定 | 値 | 説明 |
|------|-----|------|
| `style` | round | 角丸ボーダー |
| `width` | 4.0 | ボーダーの太さ |
| `hidpi` | on | HiDPI対応 |
| `active_color` | `0xc0ff00f2` | アクティブウィンドウの色（マゼンタ） |
| `inactive_color` | `0xff0080ff` | 非アクティブウィンドウの色（シアン） |

## 起動

Aerospaceから自動起動される設定になっています。

手動で起動する場合:

```bash
borders
```
