# Aerospace

macOS用のタイルウィンドウマネージャー。i3/sway風のキーバインドで操作。

## 基本設定

| 設定 | 値 |
|------|-----|
| レイアウト | tiles（タイル） |
| ウィンドウ間隔 | 5px |
| 上部マージン | 35px（sketchybar用） |

## キーバインド

### ウィンドウフォーカス

| キー | 動作 |
|------|------|
| `Alt+h` | 左のウィンドウにフォーカス |
| `Alt+j` | 下のウィンドウにフォーカス |
| `Alt+k` | 上のウィンドウにフォーカス |
| `Alt+l` | 右のウィンドウにフォーカス |

### ウィンドウ移動

| キー | 動作 |
|------|------|
| `Alt+Shift+h` | ウィンドウを左に移動 |
| `Alt+Shift+j` | ウィンドウを下に移動 |
| `Alt+Shift+k` | ウィンドウを上に移動 |
| `Alt+Shift+l` | ウィンドウを右に移動 |

### リサイズ

| キー | 動作 |
|------|------|
| `Alt+Shift+-` | ウィンドウを縮小 |
| `Alt+Shift+=` | ウィンドウを拡大 |

### レイアウト

| キー | 動作 |
|------|------|
| `Alt+/` | タイルレイアウト切替（水平/垂直） |
| `Alt+,` | アコーディオンレイアウト切替 |

### ワークスペース

| キー | 動作 |
|------|------|
| `Alt+1-9` | ワークスペース1-9に移動 |
| `Alt+a-g` | ワークスペースA-Gに移動 |
| `Alt+Shift+1-9` | ウィンドウをワークスペース1-9に移動 |

## 連携

- **borders**: 起動時にbordersを自動起動
- **sketchybar**: 起動時に自動起動、ワークスペース変更を通知
